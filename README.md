## Project Overview
This project is a Java-based Appointment Scheduling System developed using Maven.
It allows administrators to manage appointments and users to register, log in, view available slots, book appointments, cancel appointments, and edit participant counts.

## Features
- Administrator login/logout
- User registration/login/logout
- View available appointment slots
- Book appointment
- Cancel appointment
- Edit participant count
- Admin can add appointments
- Admin can edit appointments
- Admin can delete appointments
- Admin can cancel confirmed appointments

## Technologies Used
- Java
- Maven
- JUnit 5
- Mockito
- JaCoCo

## How to Run
1. Open the project in IntelliJ IDEA
2. Configure the JDK
3. Run `Main.java`
